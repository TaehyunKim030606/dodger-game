# Dodger Game

Python과 Pygame으로 만든 장애물 피하기 게임입니다.

플레이어를 움직여 적을 피하고 아이템과 파워업을 획득하며 높은 점수에
도전할 수 있습니다.

## 실행 방법

Python 3가 설치된 환경에서 다음 명령을 실행합니다.

```bash
python3 -m pip install -r requirements.txt
python3 Main_Dodger.py
```

## 주요 기능

- 장애물 피하기와 점수 시스템
- 아이템 및 파워업
- 시작, 플레이, 최고 점수 화면
- 배경 음악과 효과음
